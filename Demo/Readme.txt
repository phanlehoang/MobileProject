Hướng dẫn cài đặt
1. Cài đặt ngôn ngữ php và setup php có thể chạy toàn cục thông qua cmd
2. Chạy file start.bat (double click)
  2.1 Nếu file không chạy, có thể mở cmd và chạy lệnh "start.bat"
  2.2 Nếu chạy bằng Windows Power Shell, chạy lệnh "start-process start.bat"
3. Khởi động demo (Khuyến khích sử dụng cách 3.1)
  3.1 Folder TicketNow bằng Android Studio và chạy trên thiết bị ảo
  3.2 Cài đặt APK trên các chương trình giả lập (Nox, Bluestack, ...)
  3.3 Nếu cài đặt trên thiết bị thật. Sau khi cài đặt và vào app, nhấn nút bánh răng ở góc phải trên màn hình sau khi nó xuất hiện
      Sau đó, thay đổi Server IP bằng IP address của máy tính (xem trong ipconfig)
      Lưu ý, cần kết nối cùng một mạng

Hướng dẫn sử dụng app
1. Đăng nhập
2. Chọn cụm rạp, ngày chiếu, thể loại, độ tuổi và nhấn tìm kiếm
3. Chọn phim
4. Chọn ngày chiếu, suất chiếu 
5. Chọn chỗ ngồi bằng cách nhấn vào chọn chỗ
6. Đánh dấu các ô chỗ muốn chọn
7. Nhấn nút chọn chỗ ngồi
8. Nhấn đặt vé và app sẽ đưa chúng ta đến lịch sử đặt vé
9. Chọn một vé để xem mã kích hoạt
10. Có thể xoá vé đã đặt bằng cách nhấn nút trừ
